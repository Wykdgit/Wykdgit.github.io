下面，是我所学到的基本东西。

#1.Myssql 初涉
1.1Mysql数据库的概念理解。它是一个关系型的数据库，与redis相比，它是存在硬盘里的，而redis是将数据存到缓存里。mysql更适合存大量的基本数据，redis读取数据的速度非常的快，因此，常常用来存一些热门数据。

1.2Mysql数据库服务的启动与停止:’NET START MYSQL’
NET STOP MYSQL

1.3Mysql数据库的登录与退出 登录：mysql -uroot -p -P3306 -h127.0.0.1
参数说明：-p password -P 端口号 -h服务器地址（如果是连接远程的mysql服务，需要连接远程的服务器地址）退出 \q;

1.4Mysql提示符的设置四种提示符：\D(完整的日期) \d(当前数据库) \u(当前的用户) \h(当前的服务器名称) 改数据库提示符 PROMPT \u@\h \d> 改成了 root@localhost 当前数据库名

1.5数据库的基本操作 数据库的创建+字符集的修改 CREATE DATABASE t1 CHARACTER SET gbk(utf8); 查看当前创建的所有数据库 SHOW DATABASES; 查看某一个数据库的基本信息 SHOW CREATE DATABASE t1;

#2.数据类型与数据表的操作

##2.1 数据类型
1.整型 TINYINT SMALLINT MEDIUMINT INT BIGINT (1 2 3 4 8)
2.浮点行 FLOAT(M,D) DOUBLE(M,D) 参数说明：M指的是小数点前后的总位数，D指的是小数点后面的位数。
3.日期时间类型：YEAR TIME DATE DATETIME TIMESTAMP
4.字符型：VARCHAR(M) 0<M<=255 TEXT ENUM(‘VALUE1’,’VALUE2’,’VALUE3’)
